# Extension Icons

Place your extension icons in this folder:

- `icon16.png` - 16x16 pixels (toolbar)
- `icon48.png` - 48x48 pixels (extensions page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Creating Icons

You can create simple placeholder icons using any image editor or online tools.

For now, you can use placeholder icons or create custom ones with your preferred design tool.

### Quick Icon Creation

Use an online tool like:
- https://favicon.io/favicon-generator/
- https://realfavicongenerator.net/
- Canva (https://www.canva.com/)

### Suggested Design

- Shield icon (representing protection)
- Blue and white color scheme
- Simple, recognizable design
- Matches the app's branding

## Placeholder Icons

If you want to test the extension without custom icons, you can:
1. Create simple solid color PNG files with the required dimensions
2. Use a basic shield or report symbol
3. Replace later with professional icons
